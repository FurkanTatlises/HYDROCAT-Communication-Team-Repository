# HYDROCAT-Haberle-me-Reposu
Bu repoda Teknofest İnsansız Deniz Aracı yarışması için kullanacağımız haberleşme mimarisinin kodları bulunmaktadır.  Bu repoda çoğunluklu olarak C++ kodları bulunmaktadır ve bu kodlar LiDAR verisi işleme, YKİ ile gemi arasındaki haberleşmeyi sağlama, kumanda kontrol ve GUI tasarımı fonksiyonlarını içermektedir.
